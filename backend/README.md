# EASY-PARKING

## AI POWERD PARKING MANAGEMENT SYSTEM

Welcome to the backend API documentation for the EASY-PARKING Parking Management System .

The Edge_detectort utilizes AI/Computer Vision to automatically detect vehicle license plates and extract registration numbers.
Once the text is extracted, the data is transmitted to this centralized API, which handles all business logic, guard management, billing, parking slot allocation, and administration.

## Base URL

All API requests must be made to the following base URL:
[text](https://api.easyparking.com/api/v1)
